Esta pasta contém o código-fonte Java do projeto.
Os arquivos estão organizados no pacote 'rotasegura'.

Arquivos:
- Main.java: Classe principal que executa o sistema e realiza testes.
- Veiculo.java: Classe base para os veículos.
- Moto.java: Especialização de Veículo para motos.
- Caminhao.java: Especialização de Veículo para caminhões.
- StatusVeiculo.java: Enum que define os estados possíveis do veículo.
