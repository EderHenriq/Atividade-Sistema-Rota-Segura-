Esta pasta contém a implementação em Python do sistema Rota Segura.

Arquivos:
- gui.py: Interface gráfica construída com Tkinter que simula o sistema de registro de rotas e manutenção.
  Este script é autônomo e contém suas próprias definições de classes (Veiculo, Moto, Caminhao).

Para executar:
python gui.py
