# Sistema Rota Segura

Este projeto contém implementações em Java e Python para o sistema de gestão de veículos Rota Segura.

## Estrutura de Pastas

- **src/rotasegura**: Contém o código-fonte Java.
- **python**: Contém a versão em Python com interface gráfica.
- **docs**: Contém a documentação e enunciado do desafio.
- **bin**: Contém os binários compilados do Java.

## Como Executar

### Java
Navegue até a pasta raiz e execute:
```bash
java -cp bin rotasegura.Main
```
(Certifique-se de que os arquivos .class estão na pasta bin e estrutura correta)

### Python
Navegue até a pasta python e execute:
```bash
python gui.py
```
